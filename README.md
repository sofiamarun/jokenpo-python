# Jokenpô (Pedra, Papel e Tesoura) - Automação em Python

Script desenvolvido para simular o clássico jogo "Pedra, Papel e Tesoura" contra o computador. O projeto foca na utilização de bibliotecas nativas do Python para gerar aleatoriedade e na implementação de lógica condicional composta.

## Como funciona
1. O usuário escolhe sua jogada via entrada de dados.
2. O sistema utiliza a biblioteca `random` para escolher uma jogada aleatória para o computador.
3. O algoritmo de validação compara as escolhas e determina o vencedor com base nas regras do jogo.

## Aprendizados
- Importação e uso de bibliotecas (`random`).
- Lógica de decisão aninhada.
- Tratamento de input de usuário.
